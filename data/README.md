# Data Folder

Place the raw MetroPT-3 dataset file in this folder and rename it to:

```text
MetroPT3.csv
```

Expected path:

```text
data/MetroPT3.csv
```

The original UCI download may be named `MetroPT3(AirCompressor).csv`. Rename it before running the notebook so the code can use one consistent file path.

The raw CSV file is intentionally ignored by Git and should not be committed to GitHub.
